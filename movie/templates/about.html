Movie Reviews Django + Bootstrap

A simple movie catalog with search, images, and external links. Here you will find what the project does, how it is built, and how to use it.

Go to Home Caracteristics

What is this project?

A practice web application to manage a movie list. Allows text search, displays movie posters stored in MEDIA and, if it exists, adds an external link (for example, a trailer or official website).

Caracteristics

Search

Simple form that filters by title/description and displays results in a grid of cards.

Standardized images

Posters with consistent size to maintain proportions.

External links

If a movie has a url, a 'Movie Link' button is displayed to the external resource.

How to Use

  1. Type a term in Search for Movie and press Search.
  2. Browse the results: title, description, and poster.
  3. Click on Movie Link if you want to open the associated external link.

Technical Stack

Django 4/5 Python Bootstrap 5 SQLite (dev) Pillow (images)
